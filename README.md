## 🖥️ 실행 방법

### 개발 환경

#### Client

```
cd client
npm run dev
```

#### Server

```
cd server
npm run dev
```

### 배포 환경

#### Client

```
cd client
npm run build
sudo systemctl reload nginx
```

#### Server

```
cd server
npm run prod
```
